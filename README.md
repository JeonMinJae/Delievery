# 사용될 기술 및 라이브러리
* Presentation Layer
  - Activity
  - Fragment
  - ViewModel
  - LiveData
  - GOF State Pattern

* Repository Layer
  - Remote Data Source(Retrofit2)
  - Local Data Source(Room)

* Common
  - Coroutines
  - Android X
  - Koin (for Light-Weight DI)
  - View-Binding

* Unit Test
  - JUnit4
  - Mockito
  - Koin-TestFAST CAMPUS

* Third-Party Library
  - Glide, Glide-Transformation
  - Retrofit2
  - OkHttp3
  - CameraX
  - Google Maps, Location
  - Firebase Auth, Cloud Storage
